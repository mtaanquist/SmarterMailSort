SmarterMailSort settings

LLM endpoint

OpenAI-compatible base (Ollama, OpenWebUI, OpenAI). The /v1/chat/completions path is appended automatically.

Classification

Keep this at 1 for local models to avoid oversaturating the endpoint.

Classify several emails in a single LLM request. Higher values cut round-trips and prompt overhead — a big speed-up on large folders — but need a model that reliably returns well-formed JSON. Start around 10–20 and lower it if results degrade. 1 = one request per email.

Ollama / CORS note

Thunderbird grants this extension cross-origin access to the endpoint host once you save (it requests permission for the origin you enter). Ollama additionally rejects cross-origin requests unless started with OLLAMA_ORIGINS=* (or your extension origin). OpenWebUI and OpenAI need no extra configuration.
